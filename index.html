1️⃣ Etapa 1: Variáveis

O que é uma variável?

  • É uma "caixinha" onde guardamos informações!
  • Toda variável tem: NOME = VALOR
# 📦 Criando variáveis para guardar os números
numero1 = 10
numero2 = 5

# 📦 Guardando o resultado em outra variável
resultado = numero1 + numero2

# 📺 Mostrando na tela com print()
print("=== MINHA CALCULADORA ===")
print("Número 1:", numero1)
print("Número 2:", numero2)
print("Resultado da soma:", resultado)

💡 EXPERIMENTE!

Troque os valores das variáveis numero1 e numero2 no código e veja o resultado mudar sozinho.

🚀 Etapa 3: Loops (Projeto Final)

O que é um loop?

  • É uma "repetição"! O programa fica rodando até que uma condição faça ele parar.
  • while (ENQUANTO) → repete enquanto algo for Verdadeiro (True).
# 🔁 LOOP: ficamos calculando até o usuário querer sair
continuar = True  # variável que controla o loop

while continuar:

    print("\nOperações disponíveis: +  -  *  /")
    
    # 📥 Pedindo os dados
    entrada = input("\nDigite o 1º número (ou 'sair'): ")

    # 🤔 CONDICIONAL dentro do LOOP para sair
    if entrada.lower() == "sair":
        continuar = False
        print("\nAté logo! Encerrando...")

    else:
        try:
            numero1 = float(entrada)
            numero2 = float(input("Digite o 2º número: "))
            operacao = input("Qual operação? (+, -, *, /): ")

            # 🤔 CONDICIONAL: escolhendo a operação certa
            if operacao == "+":
                resultado = numero1 + numero2
            elif operacao == "-":
                resultado = numero1 - numero2
            elif operacao == "*":
                resultado = numero1 * numero2
            elif operacao == "/":
                if numero2 == 0:
                    print("ERRO: Não é possível dividir por zero!")
                    resultado = None
                else:
                    resultado = numero1 / numero2
            
            if resultado is not None:
                print(f"\n>>> Resultado: {numero1} {operacao} {numero2} = {resultado}")

        except ValueError:
            print("ATENÇÃO: Digite apenas números válidos!")

🖥️ Projeto Final: Calculadora Visual

Aqui vemos os 3 fundamentos no mundo real!

  • Usamos tkinter para desenhar as telas e botões de verdade.
  • Utilizamos tudo o que aprendemos: variáveis (para guardar a expressão), condicionais (para decidir o que cada botão faz) e loops (tanto no `for` para desenhar os botões quanto no `mainloop()` da janela).
import tkinter as tk

# --- 1️⃣ FUNÇÃO: O QUE ACONTECE QUANDO CLICA NO BOTÃO? ---
def clique_botao(valor):
    # 🤔 CONDICIONAIS: O que o usuário clicou?
    if valor == "=":
        try:
            # Pega a expressão da tela e resolve (eval)
            expressao = tela.get()
            resultado = eval(expressao)
            
            tela.delete(0, tk.END)
            tela.insert(tk.END, str(resultado))
        except:
            tela.delete(0, tk.END)
            tela.insert(tk.END, "Erro")
            
    elif valor == "C":
        tela.delete(0, tk.END)
        
    else:
        tela.insert(tk.END, valor)

# --- 2️⃣ CRIANDO A JANELA (O Visual) ---
janela = tk.Tk()
janela.title("Calculadora - Turma Ciência")
janela.geometry("320x450")
janela.configure(bg="#2b2b2b")

# --- 3️⃣ A TELA DA CALCULADORA ---
tela = tk.Entry(janela, font=("Arial", 30), bg="#d4d4d2", fg="black", justify="right")
tela.grid(row=0, column=0, columnspan=4, ipadx=8, ipady=20, pady=15, padx=10)

# --- 4️⃣ OS BOTÕES E O LOOP ---
botoes = [
    '7', '8', '9', '/',
    '4', '5', '6', '*',
    '1', '2', '3', '-',
    'C', '0', '=', '+'
]

linha = 1
coluna = 0

# 🔁 LOOP: Para cada botão na nossa lista, desenhe ele na tela!
for botao in botoes:
    comando = lambda x=botao: clique_botao(x)
    tk.Button(janela, text=botao, font=("Arial", 18, "bold"), width=4, height=2,
              bg="#505050", fg="white", command=comando).grid(row=linha, column=coluna, padx=5, pady=5)
    
    coluna += 1
    if coluna > 3:
        coluna = 0
        linha += 1

# 🔁 O MAIOR LOOP DE TODOS: Faz a janela não fechar!
janela.mainloop()

🗺️ Resumo dos 3 Fundamentos

1️⃣

Variáveis

Guardam os dados do programa (ex: numero1, numero2, resultado).

2️⃣

Condicionais

Decidem o caminho (if/elif/else). Usamos para verificar a operação ou se o usuário quer sair.

3️⃣

Loops

Mantêm o programa vivo (while). A calculadora não fecha até você mandar!
